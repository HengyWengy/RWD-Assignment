üìö How to Use This Application

Step 1: Setup

  1. Extract ProductivityHub-Complete.zip
  2. Copy folder to web server (XAMPP, WAMP, etc.)
  3. Create database and import tasks.sql
  4. Edit db_connect.php with your credentials

Step 2: Access Application

  1. Start Apache and MySQL
  2. Open: http://localhost/ProductivityHub-Complete/
  3. Create account or login
  4. Start adding tasks!

Step 3: Explore Features

  • Dashboard: View all tasks overview
  • Tasks: Create and manage tasks
  • Goals: Track goals and progress
  • Collaboration: Chat and share tasks
  • Analytics: View statistics and charts

Responsive Design Testing

Test on different screen sizes:

  • Desktop (1920px): Full layout with all features
  • Tablet (768px): Adapted layout, collapsible sidebar
  • Mobile (375px): Touch-optimized, hamburger menu

How to test: Press F12 in browser, click device toolbar (Ctrl+Shift+M), select device
